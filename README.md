# rn-chart-desk

This package is a wrapper for creating customizable charts in react native

## Installation

```sh
npm install rn-chart-desk
```

## Usage

```js
import { RnChartDeskView } from "rn-chart-desk";

// ...

<RnChartDeskView color="tomato" />
```
